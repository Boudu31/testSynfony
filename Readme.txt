Pour lancer synfony depuis la console de PHP Storm
    - Se placer dans le dossier : cd mon_site
    - Lancer le serveur depuis la console avec : php -S localhost:8000 -t public
Le site est accesible ensuite sur localhost:8000


Pour GitHub :
Pour connaitre le statut des fichiers : git status
Pour récupérer le contenu du Git : git pull
Pour mettre à jour le git depuis les fichiers locaux :
    -git add . (pour ajouter à l'index tous les fichiers)
    -git add nomdufichier (pour ajouter à l'index un seul fichier)

    -git commit -m "message obligatoire" (créer un commit avec le contenu actuel de l'index)

    -git push (met en ligne le commit)
